
CODSOFT Artificial Intelligence Internship

This repository contains AI internship tasks completed for CODSOFT.

Tasks

Task 1 - Chatbot

A simple AI chatbot developed using Python that interacts with users and responds to basic queries.

Task 2 - Tic-Tac-Toe AI

A simple AI-based Tic-Tac-Toe game developed using Python.

Task 3 - Image Captioning

An image captioning project that generates captions for images.

Task 4 - Recommendation System

A recommendation system that suggests items based on user preferences.

Task 5 - Face Detection and Recognition

An AI application that detects and recognizes faces in images or videos using computer vision techniques.

Technologies Used

- Python
- Artificial Intelligence
- Machine Learning Concepts
- Computer Vision
